## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Security Tools

#### Slither Analyzer

```shell
brew install slither-analyzer
slither <target-contract>
```

#### Medusa Fuzzing Test

```shell
brew install medusa
medusa fuzz
```
